# Cerebro 的安装


cerebro 是一款开源的用于管理 Elasticsearch 的 Web 端管理工具。 

<!--more-->



## 工具介绍



**项目地址**

`https://github.com/lmenezes/cerebro`



**项目依赖**

Java 11 及以上版本



## 安装



**安装方式**

- 软件包安装

  下载地址：`https://github.com/lmenezes/cerebro/releases`

- Docker（推荐）

  执行命令 `docker run --name cerebro -d -p 9000:9000 lmenezes/cerebro:0.8.4`
